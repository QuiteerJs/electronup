## 简介
[electronup-app](https://github.com/TaiAiAc/electronup-app) 是一个基于 electron 、 vue3 、 Typescript 、vite 的桌面端模版，它使用了最新流行的前端技术栈，内置了一些启动项目的通用配置，开箱即用的桌面端前端解决方案，也可用于学习参考。

## 安装使用

- 环境配置
  **本地环境需要安装 Node.js 16+ 、  Git**
